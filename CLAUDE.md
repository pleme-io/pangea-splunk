Auto-generated Ruby gem providing 4 typed Splunk resource functions for the
Pangea infrastructure DSL. Generated by pangea-forge from
`splunk-terraform-resources` TOML specs.

## Build

```bash
bundle install
bundle exec rspec   # unit tests
bundle exec rake    # default task
```

## Architecture

```
lib/
  pangea-splunk.rb               # Gem entry point
  pangea-splunk/
    version.rb                   # PangeaSplunk::VERSION
  pangea/
    resources/
      splunk.rb                  # Provider registration
      splunk_apps_local/         # SplunkAppsLocal resource
      splunk_indexes/            # SplunkIndexes resource
      splunk_inputs_http_event_collector/  # SplunkInputsHttpEventCollector
      splunk_saved_searches/     # SplunkSavedSearches resource
    types/
      splunk_types.rb            # Dry::Types module for Splunk
```

## Key Types

Each resource is a `Dry::Struct` subclass with typed attributes matching the
TOML spec. Used as: `Pangea::Resources::Splunk.splunk_indexes(name: ..., ...)`.
